# Readable Logic Tests

Учебный проект по автоматизации тестирования на Python.

## Что внутри:
1. **Formatting Functions**: Преобразование технических имен функций в читаемый вид.
2. **User Search**: Фильтрация списка пользователей по имени и возрасту.
3. **Dark Theme**: Логика переключения темы сайта в зависимости от времени суток и настроек пользователя.

## Как запустить:
`python tests.py`
